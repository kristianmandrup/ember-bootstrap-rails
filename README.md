= Ember Bootstrap Rails

Add ember-boostrap to your rails/ember project :)

## Install

Gemfile

```ruby
gem 'ember-bootstrap-rails'
```

And then `bundle` :)

In your js mainfest file:

```javascript
//= require ember-bootstrap-rails/all
````

Enjoy!
